# guilherme.dev

Portfólio pessoal de Guilherme Henrique de Araújo Santos, desenvolvido com HTML, CSS e JavaScript puros.

Acesse em: [guilhermehenriquesantos.github.io](https://guilhermehenriquesantos.github.io/)

## Informações

- Design responsivo (desktop, tablet e mobile);
- Conteúdo bilíngue (PT/EN) com alternância dinâmica;
- Listagem de repositórios em tempo real via GitHub API, com cache local e fallback offline;
- Animações com respeito a `prefers-reduced-motion`;
- Acessibilidade: navegação por teclado, skip link, atributos ARIA e HTML semântico;
- SEO: meta tags Open Graph, dados estruturados (JSON-LD) e URL canônica.

## Estrutura

```text
index.html              Página única do portfólio
assets/css/style.css    Estilos (tokens de design, seções e responsividade)
assets/images/          Imagens e ícones
javascript/functions.js Idioma, menu móvel, GitHub API, typewriter e interações
```
